Enregistrement de la configuration générale du jour
Pour enregistrer votre configuration du jour (en la définissant), il suffit de cliquer sur le bouton “Passer à la configuration suivante” en cochant la case . Cette action mémorise, pour la prochaine séance, la configuration qui sera utilisée aujourd'hui.
