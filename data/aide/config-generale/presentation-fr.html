Configuration générale des exercices
La “configuration générale” de l'exercice consiste à définir trois paramètres pour varier les plaisirs lors de chaque séance de travail :
  • Le sens des exercices ;
  • La suite harmonique utilisée ;
  • L'Ordre des exercices.
Enregistrement de la configuration générale du jour
Pour enregistrer votre configuration du jour (en la définissant), il suffit de cliquer sur le le volant de la configuration générale des exercices pour choisir la configuration voulue, puis de cliquer sur le bouton d'enregistrement. Cette action mémorise, pour la prochaine séance, la configuration qui sera utilisée aujourd'hui.
Noter que lorsque vous demandez la confection d'une séance de travail, vous avez une option (case à cocher) pour passer automatiquement à la configuration suivante.
