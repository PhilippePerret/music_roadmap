Base de données d'exercices
La base de données d'exercice contient des exercices à introduire dans sa feuille de route, qui contiennent toutes les données nécessaires (types de difficultés, tempi, etc.).
Pour choisir un exercice, il suffit d'ouvrir la base de donnée et de choisir l'exercice, qui est classé soit par auteur soit par recueil (lorsque le recueil contient différents auteurs.)
Cette base de données grossit de jour en jour. Phil n'ayant pas de priorité particulière, n'hésitez pas à lui soumettre les exercices que vous aimeriez voir ajouter le plus tôt possible.
