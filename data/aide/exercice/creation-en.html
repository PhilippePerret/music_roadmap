Add exercices to roadmap
Required: to add exercices, you have to be signed in and to have a roadmap opened ( see create a roadmap or open a roadmap).
When a roadmap is opened, you can see the “New Exercice” button.
If this button is visible, you can click on it to initiate a new exercice. A form opens where you can define the characteristics of the exercice.
Note the first button, at the top of the form. When you click on it, you reach the Exercices Data Base of “Music Roadmap”. You just have to choose exercices in this database to add them to your roadmap.
(this database grows up day after day, so don't hesitate to submit a request to Phil for a new exercice, via the contact button. This exercice will be added in priority.)
If you want to know how to add exercices from the database, see Add an exercice from the Exercices Data Base. Otherwise, if you want to add an exercice of yours, read the following.
Field descriptions
  • The “Collection” field defines the book title from which the exercice or the study is extracted (p.e. “Study op. 25”). You can omit this value.
  • The “Title” field contains… the title of the exercice, which just can be “n°12” if its collection is defined. This title is required.
  • The “Author” field defines the author of the exercice (e.g. “Chopin”).
  • Under these three general fields you find two checkboxes:
    • The “Obligatory” checkbox lets you define that this exercice must always be worked, when you ask Music Roadmap to prepare a work session for you.
    • The “Linked to next” checkbox lets you specify that this exercice must be link to the next exercice. That means that if you play the exercices without stopping, Music Roadmap will share to total working time between all exercices linked (yes, Music Roadmap records your working time)
  • Under these checkboxes you can define the difficulties of the exercice. Open the difficulty list with the “modify…” button then check every difficulty the exercice focuses on.
    Note that this definition is very important if you want Music Roadmap to show you usefull reports. Thanks to the difficulties definition, Music Roadmap can tell you the working time on each difficulty for the day or a period of time (by default: the current month). So you can know exactly how much time you focused on each difficulty.
  • The “Tempi” row contains three pop-up. You can define:
    • The current tempo for the exercice;
    • The slowest tempo (e.g., in the “Hanon”, min tempo of the first exercice should be set to 60);
    • The fastest tempo (e.g., in the “Hanon”, max tempo of the first exercice should be set to 108).
  • The “Tone” pop-up lets you define the tone of the exercice, if any. Note that if this tone is not defined, so it is an “harmonic” exercice for Music Roadmap, and a new tone will be provided at each new work session.
  • The “Suite” pop-up lets you define the “Harmonic Suite” of the exercice. Click on the "?" which appears when you move your mouse over the pop-up to have more details.
  • The “Measure number” lets you define the measure count for the exercice. This data is usefull to prepare work session for a fixed time. Count ALL the measures, i.e. with repetitions.
  • Then define the “Number of beats per measure”.
    Note that Music Roadmap can calculated the duration of the exercice according to the last three information. Set them with care.
    But Music Roadmap is smart, so it also uses your real working time on the exercice. That means that the more you work with Music Roadmap, the more precise is the calcul of the tailored work session.
  • You can add an image (excerpt of the score) with the Image field. This image is not settable yet, but you can send your images to Phil to add them to your roadmap.
  • Finally, you can add a note with the “Note” field. Maybe a recommandation of you professor. You can update this field as soon as it's necessary, with the “edit” button of the exercice (in the exercice listing).
  • Nothing right new about the “Import” row. You have to have several roadmaps to use it.
You can now create the exercice clicking the Il ne vous reste plus qu'à créer l'exercice en cliquant sur “Save this new exercice” button.
Congratulations!
You can repeat these operations to create other exercices (at least three or four to follow this manual), then you'll can see the running exercices part or the Preparation of a work session.
