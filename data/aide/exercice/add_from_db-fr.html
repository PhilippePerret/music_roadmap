Ajouter un exercice depuis la base de données
Pré-requis
Pour pouvoir ajouter un exercice depuis la base de données, vous devez :
  • Être inscrit et être identifié (inscription ou identification) ;
  • Avoir créé une feuille de route (création d'une feuille de route) ;
  • Avoir ouvert la feuille de route dans laquelle vous voulez ajouter l'exercice (chargement d'une feuille de route).
Importation de l'exercice depuis la base de données
  • Cliquez sur le bouton de création d'un nouvel exercice ;
    -> Le formulaire de nouvel exercice s'ouvre.
  • Cliquez sur le bouton pour atteindre la base de données ;
    -> La page d'accueil de la base de données s'ouvre.
  • Cliquez ensuite sur le nom du compositeur de l'exercice ou le recueil auquel cet exercice appartient ;
    -> La liste des exercices ou la liste des recueils apparait.
  • Si vous avez choisi un compositeur, cliquez sur le nom du recueil auquel appartient l'exercice.
  • Cochez l'exercice à ajouter et recommencer l'opération pour chaque exercice que vous voulez ajouter (notez que même si vous refermez un recueil, les exercices cochés restent choisis) ;
  • Cliquez enfin sur le bouton d'ajout des exercices pour ajouter tous les exercices choisis à votre feuille de route.
Optionnel
Vous pouvez maintenant classer ces exercices (cf. Classement des exercices) et/ou régler leur tempo initial.
