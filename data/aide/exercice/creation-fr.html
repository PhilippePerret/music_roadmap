Ajout d'exercices à la feuille de route
Avant de pouvoir ajouter des exercices, vous devez impérativement avoir créé une feuille de route ou avoir ouvert une de vos feuilles de route.
C'est seulement lorsqu'une de vos feuilles de route est ouverte que le bouton Créer exercice apparait.
Si ce bouton est visible, vous pouvez donc cliquer dessus pour initier un nouvel exercice . Un formulaire s'ouvre devant vos yeux ébahis. C'est le formulaire où vous allez définir les “paramètres” de votre nouvel exercice.
Notons pour commencer le premier bouton, tout en haut, qui va énormément vous simplifier la vie. Il vous permet d'avoir accès à une base de données d'exercices qu'il suffit de cliquer pour les ajouter à votre feuille de route.
Bien entendu, encore faut-il que cet exercice se trouve dans la Base de Donnée du site. Mais elle grossit de jour à jour, et n'hésitez pas à soumettre une demande par le biais du formulaire de contact par mail.
Si vous voulez tout de suite savoir comment ajouter des exercices depuis la base de données, alors rejoignez l'article Ajouter un exercice de la base de données des exercices. Dans le cas contraire, si vous voulez apprendre à faire un exercice particulier, poursuivez cet article.
Description des champs
  • Le champ “Recueil”contient le titre du livre d'où est issu l'exercice ou l'étude (p.e. “Études op. 25”). Cette valeur peut être omise.
  • Le champ “Titre” contient… le titre de l'exercice, qui peut se résumer à un “n°12” si le recueil est défini. Ce titre est obligatoire.
  • Le champ Auteur définit l'auteur de l'exercice, en d'autres termes : le compositeur ;-).
  • Sous ces trois champs généraux se trouve deux cases à cocher dont vous n'aurez peut-être pas besoin tout de suite. Je les explique brièvement :
    • La case “Obligatoire” permet de définir un exercice qu'il faudra impérativement ajouter à une session de travail, lorsque vous demandez à Feuille de Route de vous confectionner une séance de travail.
    • La case “Lié au suivant” permet de lier cet exercice au suivant, c'est-à-dire que vous les jouerez sans vous arrêter, et Feuille de Route répartira le temps total de travail entre ces deux exercices (car Feuille de Route mémorise votre temps de travail pour vous proposer une vision plus précise de votre travail sur le court et le long terme)
  • La rangée “Tempi” contient trois menus permettant de régler, dans l'ordre :
    • Le tempo où vous allez commencer par jouer cet exercice (peut-être le plus lent, ou légèrement au-dessus), grâce au menu Tempo ;
    • Le tempo le plus lent de l'exercice grâce au menu Tempo minimal (p.e., dans le Hanon, le premier exercice doit se travailler à 60 à la noire. Il faudrait donc régler ici “60”) ;
    • Le tempo le plus rapide de l'exercice grâce au menu Tempo maximal (p.e. dans le Hanon, le premier serait réglé à “108”).
    Le saviez-vous ?… Quand vous vous trouvez sur un menu tels que ceux-là, vous pouvez taper les chiffres sur votre pavé pour choisir le nombre voulu (sur MAC, en tout cas).
  • Le menu “Suite” permet de définir la “Suite harmonique” qui est peut-être appliqué à l'exercice. Pour le moment, vous pouvez passer cette information. Vous pourrez cliquer sur le “?” à côté d'elle lorsque vous serez prêt à en savoir plus.
  • Vous pouvez ajouter une image (partition) de votre exercice grâce au champ image suivant. Cette image est pour le moment désactivée, mais vous pouvez contacter Phil directement pour lui demander d'ajouter des images que vous lui transmettrez.
  • Enfin, vous pouvez ajouter une note sur l'exercice grâce au . Cette note peut être une recommandation pour bien exécuter l'exercice ou le morceau, par exemple. Ça peut être aussi les recommandations de votre professeur. Au cours de votre travail, vous pourrez actualiser ce champ à mesure de la progression et des conseils que vous pourrez recevoir.
  • Nous pouvez passer la rangée “Import”, que vous pourrez aborder lorsque vous aurez d'autres feuilles de route, ou que vous connaitrez mieux l'application.
En définitive, les seuls champs qui sont obligatoirement à définir pour créer un exercice sont : le titre de l'exercice et le tempo maximum (l'application refuse que les deux tempi, minimum et maximum, soient équivalents — elle a de ses caprices, parfois…).
Il ne vous reste plus qu'à créer l'exercice en cliquant sur le bouton “Créer l'exercice”.
Bravo à vous !
