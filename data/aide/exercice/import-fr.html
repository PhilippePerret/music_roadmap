Importer des exercices
Présentation
Grâce à la fonction d'importation, vous pouvez importer dans une nouvelle feuille de route des exercices appartenant à d'autres feuilles de route.
Comment faire ?
  • Vous devez tout d'abord créer une nouvelle feuille de route si nécessaire ;
  • Ouvrez la boite d'édition des exercices en cliquant sur le bouton “Nouvel exercice” ;
    -> La boite d'édition doit s'ouvrir.
  • Indiquez dans le champ des chemins d'exercices les “paths” des exercices (cf. Définir les chemins des exercices ci-dessous) ;
  • Cliquer sur le bouton “Importer” pour importer le ou les exercices spécifiés.
    -> Ils doivent se charger dans votre feuille de route.
Vous aurez peut-être besoin de recharger la page pour que les images de ces exercices apparaissent.
Définition des chemins des exercices
Un “chemin d'exercice” est spécifié par :
  • Le nom de sa feuille de route (par exemple “exemple”) ;
  • Son mot de passe (p.e. “motdepasse”) ;
  • Son identifiant (toujours un nombre)
Chacun de ces éléments est séparé de l'autre par une balance (“/”). Par exemple : feuille_de_route/mot_de_passe/125.
Dans le champ d'importation d'exercices, vous pouvez spécifier plusieurs exercices en les séparant par des virgules, sans espaces :
feuille01/password/12,ma_map/moncode/24,feuilleessai/code/32.
Dans l'exemple ci-dessus, on importera l'exercice #12 de la feuille “feuille01” qui est protégée par le mot de passe “password”, l'exercice #24 de la feuille “ma_map” protégée par “moncode” et l'exercice #32 de la feuille “feuilleessai” protégée par “code” (de très mauvais mots de passe, inutile de vous le dire ;-)).
Obtenir le chemin d'un exercice
Il y a une chose toute simple à faire pour obtenir le chemin d'un exercice sans vous tromper et de façon très rapide :
  • Dans une autre page, rejoignez Feuille de Route Musicale, ouvrez votre feuille de route qui contient l'exercice à importer ;
  • Cliquez sur le chiffre lègèrement bleu, entre crochets, qui se trouve sous le titre de l'exercice, tout à gauche 
    -> L'application ouvre un message (en haut) contenant le chemin de l'exercice ;
  • Placez votre curseur dans ce chemin du message.
    -> Tout le chemin devrait se sélectionner ;
  • Copiez-collez le dans votre feuille de route qui doit importer l'exercice, sans oublier de le séparer d'une virgule si vous avez déjà indiqué d'autres chemins d'exercices.
