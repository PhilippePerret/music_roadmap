Création des exercices
Le saviez-vous ?… Lorsque vous arrivez sur la page de cette application, le métronome est déjà en train de battre. Cela peut devenir très vite agaçant. Pas de panique : pour l'arrêter, il suffit de cliquer dessus ! :-)
Avant de pouvoir créer des exercices, vous devez impérativement avoir ou avoir . Dans tous les cas, votre feuille de route doit être ouverte.
Dans le cas contraire, de toute façon, le ne serait pas visible dans votre interface.
Si ce bouton est visible, vous pouvez donc cliquer dessus pour initier un nouvel exercice . Le formulaire s'ouvre devant vos yeux ébahis.
Description des champs
  • Le contient le titre du livre d'où est issu l'exercice ou l'étude (p.e. “Études op.25”). Cette valeur peut être omise.
  • Le contient… le titre de l'exercice, qui peut se résumer à un “n°12” si le recueil est défini. Ce titre est obligatoire.
  • Nous pouvez passer la rangée “Import”, que vous pourrez aborder lorsque vous aurez d'autres feuilles de route, ou que vous connaitrez mieux l'application.
  • Le définit l'auteur de l'exercice, en d'autres termes : le compositeur ;-). Cette valeur peut être omise.
  • La rangée “Tempi” contient trois menus permettant de régler, dans l'ordre :
    • Le tempo où vous allez commencer par jouer cet exercice (peut-être le plus lent, ou légèrement au-dessus), grâce au  ;
    • Le tempo le plus lent de l'exercice grâce au (p.e., dans le Hanon, le premier exercice doit se travailler à 60 à la noire. Il faudrait donc régler ici “60”) ;
    • Le tempo le plus rapide de l'exercice grâce au (p.e. dans le Hanon, le premier serait réglé à “108”).
    Le saviez-vous ?… Quand vous vous trouvez sur un menu tels que ceux-là, vous pouvez taper les chiffres sur votre pavé pour choisir le nombre voulu (sur MAC, en tout cas).
  • Le permet de définir la “Suite harmonique” qui est peut-être appliqué à l'exercice. Pour le moment, vous pouvez passer cette information.
  • Vous pouvez ajouter une image (partition) de votre exercice grâce au suivant.
  • Enfin, vous pouvez ajouter une note sur l'exercice grâce au . Cette note peut être une recommandation pour bien exécuter l'exercice, par exemple. Au cours de votre travail, vous pourrez actualiser ce champ à mesure de la progression et des conseils que vous pourrez recevoir.
En définitive, les seuls champs qui sont obligatoirement à définir pour créer un exercice sont : le titre de l'exercice et le tempo maximum (l'application refuse que les deux tempi, minimum et maximum, soient équivalents — elle a de ses caprices, parfois…).
Il ne vous reste plus qu'à créer l'exercice en cliquant sur .
