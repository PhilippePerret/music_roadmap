Inscription sur Feuille de Route Musicale
  • Cliquez sur le bouton d'identification et d'inscription ;
    -> Le formulaire d'identification s'ouvre.
  • Cliquez sur le bouton d'inscription ;
    -> Le formulaire d'inscription s'ouvre.
  • Remplissez tous les champs (notez particulièrement le menu “Instrument”, qu'il faut renseigner avec soin puisqu'il vous donnera accès à la base de données des exercices de cet instrument) ;
  • Cliquez enfin sur le bouton “S'inscrire”.
Vous devriez recevoir une confirmation par mail, avec votre mail et votre mot de passe. Gardez-le précieusement puisque ces informations vous permettront de vous identifier une prochaine fois.
