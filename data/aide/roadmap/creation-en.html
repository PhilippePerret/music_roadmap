Create your musical roadmap
Note that to be able to create a roadmap, you have to be signed in (or sign up). To do so, use the Signin/Signup button.
Set the name of your roadmap
Create a new roadmap is simple as "Hello". Once signed in, you just have to have a name for it.
You can proceed as follows:
  • Sign in (or up) if needed;
  • Fill the Roadmap Name field with the name you've choosen (note your spaces will be turn into underscores, don't be afraid ;-));
  • Hit the tab key to get out the field, so the buttons roadmap can appear.
N.B.: If this page is over the text field, hold down the mouse on this box bar and move the box.
Really create it?
Now, you just have to click the Create button.
