Exemple de feuille de route musicale
Ouvrir la feuille de route exemple
Pour afficher un exemple de feuille de route et voir les premières fonctionnalités utiles de l'application (rappel : vous pouvez déplacer la fenêtre de cette aide en la prenant par sa bande supérieure ;-) ) :
  • Inscrivez “exemple” (tout en minuscule) dans le champ “FEUILLE DE ROUTE” ;
  • Inscrivez “exemple” (tout en minuscule aussi) dans le champ “MOT DE PASSE”
  • Tapez la touche Tabulation ou cliquez n'importe où dans la page pour faire apparaitre le bouton “Ouvrir”
  • Cliquez ce bouton “Ouvrir” pour afficher la feuille de route musicale exemple.
